Sobre o Projeto

Esse projeto foi desenvolvido com o ojetivo de praticar as linguagens HTML 5 e CSS 3, além de introduzir o uso da lógica de programação em Javascript em algumas páginas.

O objetivo era apresentar um portfólio iniciante diferente de qualquer outro, visando um HTML 5 semântico, mas caprichando no desing da página com CSS 3.

Nele você encontrará minha Biografia completa, tecnologias que domino e as que estou estudando, as ferramentas que uso no dia-a-dia, minhas certificações, projetos e página de contato com links direto para as minhas redes de contato.
